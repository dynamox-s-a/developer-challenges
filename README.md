## Iniciar Repositório
1. Após clonar e acessar o repositório, instale as dependências e inicialize o projeto
  * Instale as dependências:
    * `yarn`
  * Inicialize o projeto:
    * `yarn dev` (A aplicação será iniciada em http://localhost:3000/)
